# 数据多源互查 (Cross-source verification)

用户原话（round 5）：
> 对我们已有的数据逐年一一对照，多来源互查。我们的模型确实有用，但是是为了以年度估算为参考，普查为基础，相互修正得到的。

## 1. 模型的数据使用结构 — 确认

| 角色 | 来源 | 我们用它做什么 |
|---|---|---|
| **基础 (basis)** | 五次人口普查（1982/1990/2000/2010/2020）公报 | (a) 提供 1982 起始金字塔；(b) 提供 5 锚的 q(x) 男女分龄死亡率（CensusLifeTables）；(c) 提供 5 锚 SRB / MAFM / e0 |
| **参考 (reference)** | NBS《中国统计年鉴》历年 + 民政部公报 + WHO GHO | (a) 每年总人口（年末口径）作 PopulationAlignment 锚；(b) 每年出生 / 死亡 / 结婚 / 离婚 / 婚率 作硬约束 |
| **相互修正** | (1) 普查年的 SRB / MAFM / e0 → 用普查值直接覆盖年鉴中间插值；(2) 年鉴的年末总人口 → 缩放 CCM 投影金字塔总量；(3) 未实现：CCM 项目-and-调整使到 2020 普查时点的形状与普查 enumeration 一致 | 见 docs/AUDIT.md §1 |

**符合用户描述**——但 **mutual correction 是非对称的**：
- 普查 → 年鉴方向：通过 q(x) 影响 CCM 投影
- 年鉴 → 普查方向：通过 PopulationAlignment 缩放总量
- 真正双向迭代（WPP §I.A 描述的 project-and-adjust + migration 残差吸收）**未实现**，是 round 5+ 的 deferred 工作。

## 2. 已知交叉源差异

### 2.1 出生人口 — 7普回溯修正 vs 原 NBS 年鉴 vs 易富贤估计

| 年份 | 原 NBS 年鉴 (我 CSV) | 7普回溯修正 (NBS 2021-05) | 易富贤估计 | 差异 |
|---|---|---|---|---|
| 2016 | 1786 万 | **1883 万** | ~1300 万 | +97 / -486 |
| 2017 | 1723 万 | **1765 万** | ~1300 万 | +42 / -423 |
| 2018 | 1523 万 | ~1623 万* | ~1240 万 | +100 / -283 |
| 2019 | 1465 万 | ~1565 万* | ~1100 万 | +100 / -365 |
| 2020 | 1200 万 (= 七普 enumeration) | 1200 万 | ~1003 万 | 0 / -197 |

*  2018/2019 7普回溯具体值未单独公布，按 "两次普查间相关年份平均修订 +100 万/年" 推算（[知乎讨论](https://zhuanlan.zhihu.com/p/1921297728215967036)）。

**当前 CSV 取**：原 NBS 年鉴值（"数值采纳"原则；NBS 是最广泛接受的下限）。

**已知不准方向**：
- 7普 修正方向（+100/年）暗示年鉴可能略低估；但
- 易富贤等独立学者方向（-30~50%）暗示年鉴可能严重高估
- 二者矛盾→当前模型走 NBS 中间路线，**右栏"日志下方"citation 应标记此 caveat**（已在 data/sources.md 提及）

### 2.2 总人口 — 普查 enumeration vs 年鉴年末

| 年份 | NBS 年末估计 (我 CSV) | 普查 enumeration | 差异 |
|---|---|---|---|
| 1982 | 101654 万 | 100818 万 (三普 7-1) | +836 |
| 1990 | 114333 万 | 113368 万 (四普 7-1) | +965 |
| 2000 | 126743 万 | 126583 万 (五普 11-1) | +160 |
| 2010 | 134091 万 | 133972 万 (六普 11-1) | +119 |
| 2020 | 141212 万 | 141178 万 (七普 11-1) | +34 |

普查时点（7-1 或 11-1）与年末（12-31）口径有 0.3-0.8% 的差距，主要原因：
- 普查时点早于年末半年到一年的时间差
- 户籍人口（年鉴用公安户籍 + 抽样）vs 常住人口（普查用实地登记）的细微差

**当前 CSV 取**：NBS 年末口径（与逐年序列连续）。
**PopulationAlignment 用它**：保证显示总人口与 NBS 公开发布一致。

### 2.3 死亡人口 — NBS 年鉴 vs 国际超额死亡估计

| 年份 | NBS 年鉴 (我 CSV) | 国际估计 (例) | 差异 |
|---|---|---|---|
| 2020 | 998 万 | WHO 估计 +0 万 vs NBS | 持平 |
| 2021 | 1014 万 | WHO 持平 | 持平 |
| 2022 | 1041 万 | ~1100 万（含 2022-12 COVID 解封超额起点） | -60 |
| **2023** | **1110 万** | IHME / Airfinity 估 +200~300 万 超额（2022-12 至 2023-01 集中） | -90~-200 |
| 2024 | 1093 万 | 接近持平 | 持平 |
| 2025 | 1131 万 | 数据近 (2026-02) | — |

**2023 是已知 NBS 显著低估年**，但国际估计的方差也大。当前 CSV 取 NBS 值并在 sources.md 已注 caveat。

### 2.4 出生性别比 — 普查 / 1% 抽样 vs 独立估计

| 年份 | NBS 锚（普查/1%）| 上限估计（包含女婴瞒报） | 差异 |
|---|---|---|---|
| 2000 | 116.9 (五普) | ~119 | -2 |
| 2010 | 117.94 (六普) | ~120 | -2 |
| 2020 | 111.3 (七普) | ~113 | -2 |

女婴瞒报使 NBS 系列**长期偏低**于真实 SRB ~1-2 单位。当前 CSV 取 NBS 直接值。

### 2.5 e0 — 普查 vs WHO

| 年份 | NBS / 普查 (我 CSV) | WHO GHO |
|---|---|---|
| 1990 | 68.55 (四普) | 69.1 |
| 2000 | 71.40 (五普) | 71.7 |
| 2010 | 74.83 (六普) | 75.1 |
| 2020 | 77.93 (七普) | 78.0 |

差异 < 0.5 年，可忽略。WHO 偏高一点，原因可能是普查后的 ex-ante 估计 vs 普查派生生命表。

### 2.6 结婚 / 离婚 — 民政部 2003+ vs 2003 前？

民政部系统性公布起 2003（部分 1985-2002 数据散见公报但不系统）。当前 CSV 仅 2003+。
1985-2002 期间的"粗结婚率"在 LookupOrInterp 中外推为 2002 值 (6.10‰)——**已知偏低**，实际 1985 应 ~9-10‰。

未来工作：补 1985-2002 民政部历史数据，至少补 1987 / 1995 两个锚。

## 3. 总结判断

| 数据列 | 准确度 | 当前 CSV 是否需要更换值 |
|---|---|---|
| 总人口年末 (1978-2025) | ★★★★★ | 不动 |
| 出生人口 1978-2014 | ★★★★ | 不动 |
| 出生人口 2015-2019 | ★★★ | 不动；caveat 记入（7普 +100/年 修正可选） |
| 出生人口 2020-2025 | ★★★★★ | 不动 |
| 死亡人口 1978-2022 | ★★★★ | 不动 |
| 死亡人口 **2023** | ★★ | 不动；caveat 已记（国际估计上限更高） |
| 死亡人口 2024-2025 | ★★★★ | 不动 |
| SRB 普查 1982-2020 | ★★★★★ | 不动 |
| SRB 非普查年 | ★★ | 不动（插值） |
| MAFM 普查 1982-2020 | ★★★★ | 不动 |
| MAFM 非普查年 | ★★ | 不动（插值） |
| e0 普查 1982-2020 | ★★★★★ | 不动 |
| e0 非普查年 / WHO | ★★★ | 不动 |
| 结婚率 2003-2025 | ★★★★ | 不动 |
| 结婚率 1985-2002 | ★ | 标记为外推，未来补 |

**结论**：现有 CSV 数值与"以年度估算为参考、普查为基础、相互修正"的设计意图一致。**没有需要立即更改的值**，但 2023 死亡、2015-2019 出生、2002 前结婚率三个区域是已知的弱点，记入 deferred 工作清单。
