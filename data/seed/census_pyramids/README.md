# 普查年金字塔数据

每个 `pyramid_YYYY.csv` 描述一次普查年的年龄结构。

列：`age, male, female`（单位：**人**，不是万人）

如果某个 YYYY 没有对应 CSV，程序会回退到 `Core/Data/CensusSeedPyramids.cs` 里的内置近似值（按 5 岁组均匀展开到单岁；总人口与官方公报吻合但单岁级精度有限）。

要替换为真实数据：从 NBS 普查公报抓取（如 2020 七普 [统计局公告](http://www.stats.gov.cn/sj/pcsj/rkpc/d7c/)），生成 0..100 单岁 × M/F 的 CSV 投入此目录即可，下次启动自动覆盖。

附注：性别比在 5-9 / 10-14 / 15-19 等青少年组也偏高（出生时偏高 + 男婴存活率劣势的合力），是结构性事实，不是数据错误。
