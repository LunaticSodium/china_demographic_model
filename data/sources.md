# 种子数据来源与口径

## 总体立场

参考主项目 `PHILOSOPHY.md` —— **政府 / 半官源默认不信**，但出生 / 婚姻这类有较多独立交叉源、且数据本身用于"国家治理"而非"对外宣传"的指标，可视作**实证下限**而非真相终点。

> 即便是"最内的账本"也未必保障真实性。本工具的 `LockToHistory` 模式提供"以公开数据为约束"的紧模型基线；解除锁定后可自由反事实。

## 各文件

| 文件 | 来源 | 已知问题 |
|---|---|---|
| `births_yearly.csv` | NBS《中国统计年鉴》历年 / 人口与计划生育公报 | 2017-2019 间多个独立学者认为高估（梁建章 / 何亚福 / 易富贤等），2020+ 接近真实 |
| `deaths_yearly.csv` | NBS《中国统计年鉴》 | 2023 公布 1110 万 —— 包含 2022-12 至 2023-01 COVID 解封超额，但与国际学者估计上限相比偏低。建议作为下限。 |
| `sex_ratio_at_birth.csv` | 历次普查 + 1% 抽样调查 | 仅普查年份可靠；中间用线性插值；女婴瞒报使官方 SRB 长期偏向"实际更高"方向 |
| `marriage_rate.csv` | 民政部《社会服务发展统计公报》 | 2003 前数据稀疏；2020+ 受疫情登记延迟；离婚冷静期（2021）实施使 2021+ 结婚率下行加速 |
| `mean_age_first_marriage.csv` | 历次普查公报 | 仅 5 个数据点；2010 → 2020 跳变（女 +4 岁）显著反映 80 后晚婚 |
| `life_expectancy.csv` | 历次普查直接公布 + WHO GHO / NBS 非普查年估计 | 普查年 e0 可靠度高；非普查年 ±0.5 年波动；2020-2023 COVID 影响有争议。 |
| `total_population_yearbook.csv` | NBS《中国统计年鉴》"年末总人口" | 与"普查时点人口"是两个口径；本工具用作 `PopulationAlignment` 的目标值。 |
| `marriages_yearly.csv` | 民政部《社会服务发展统计公报》 | 仅 2003 起系统公布；单位万对。 |
| `divorces_yearly.csv` | 民政部《社会服务发展统计公报》 | 2021 离婚冷静期实施后陡降；单位万对。 |
| `census_pyramids/pyramid_*.csv` | 历次普查公报 5 岁组 → 均匀展开 | 单岁精度有限；80+ 用指数衰减分配；如已知更细，请替换为真实 CSV |

## 公开数据 vs "真实数据" 的认识论位置

参 `latex/sections/stage1.tex` 的内部数据 caveat 速记：

- 自下而上的层级汇报 → 每一层都有美化激励；
- 维稳叙事的顶层需求 → 需要"可用"数据而非"真实"数据；
- 政治正确的语义过滤 → 某些类别（"权贵资产阶级"等）连命名都被禁止；
- 跨级核验缺位 → 除非高层有具体执政需要，否则没有动力让各级数据相互校验。

本工具用"以观测数据为紧约束、反事实修改观察偏离"代替"以观测数据为真相"。

## 建议下一步交叉验证源

- **WPP (UN World Population Prospects)**：联合国对中国出生数 / TFR / e0 的独立估算。
- **Wittgenstein Centre Human Capital Data**：欧盟资助的人口预测数据库。
- **梁建章 / 育娲人口报告**：民间反公开数据偏差的独立研究。
- **CFPS / CGSS 微数据**：可自构同口径序列（PHILOSOPHY 中已记 Path B 路径）。
